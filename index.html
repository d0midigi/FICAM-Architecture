Documentation Hub

A brutalist, high-contrast documentation site built with pure HTML, CSS, and JavaScript. No frameworks. No bullshit.

Overview

This documentation system is designed for developers who want something that works out of the box. It features a responsive sidebar, multi-level dropdown navigation, and a component library you can steal for your own projects.

Every piece of this site is contained in a single HTML file. Copy it, modify it, deploy it. The sidebar automatically generates links based on the headers present on the current page.

Pro tip: Click any dropdown link to navigate between pages. The sidebar will rebuild itself dynamically based on the page content.

Usage

To add a new page, duplicate a <section class="page-section"> block, give it a unique ID, and add a link in the dropdown. The sidebar script will handle the rest.

Note: Make sure every page has headers with id attributes so the sidebar can link to them.

Customization

Tweak the CSS variables at the top of the <style> block to change colors, spacing, and border radius. The accent colors are #ff2a6d (pink) and #05d9e8 (cyan).

Technical Details

The page switching logic is about 10 lines of JavaScript. It adds and removes the active class from page sections. That is it. No React. No Vue. No 500KB bundle.

javascript 
function showPage(pageId) {
  document.querySelectorAll('.page-section').forEach(s => s.classList.remove('active'));
  document.getElementById('page-' + pageId).classList.add('active');
  buildSidebar();
}

What is this documentation system?
It is a single-file documentation site built with HTML, CSS, and vanilla JavaScript. No frameworks, no build steps, no nonsense.
How do I add a new page?
Copy an existing <section class="page-section"> block, give it a new ID, update the content, and add a dropdown link pointing to it.
Can I change the colors?
Absolutely. Edit the CSS variables at the top of the style block. The main accent colors are --accent (pink) and --accent-2 (cyan).
Is it responsive?
Yes. It works on mobile, tablet, and desktop. The sidebar hides on small screens and the grids collapse to single column.
